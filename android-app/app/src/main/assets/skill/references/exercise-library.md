# Exercise Library

Use movement patterns and equipment options instead of forcing one exercise. Choose exercises that the user can perform with stable technique, target sensation, and tolerable joints.

If the user provides equipment photos, exercise photos, or video frames, use `visual-analysis-and-food-estimation.md` for equipment identification and exercise form analysis before choosing cues or substitutions. Ask for load, reps, set number, RIR/RPE, pain, and target muscle so the visual observation can be connected to the session log.

## Exercise Visual Guide

When outputting exercises for a beginner or a user who may not recognize professional exercise names, include a concise visual guide cue that maps the name to an equipment/action category. Use the same visual IDs, Chinese equipment labels, simple instance diagrams, action path cues, beginner recognition cues, equipment markers, example movements, common movements, and look-for cues the Android app can display:

- VG-01 Smith machine (史密斯机): fixed bar path for guided presses, squats, or rows. Unified instance diagram: fixed bar between two vertical rails. Action path cue: the bar travels up and down on rails while the bench or stance stays under the track. Beginner cue: 先找两根竖轨和一根固定杠；上斜推、深蹲、划船都沿这条轨道完成。 Equipment markers: two vertical rails, fixed guided bar, safety hooks. Example movement: Incline Smith Press. Common movements: Incline Smith Press, Smith Squat, Smith Row. Look-for cue: Look for two rails and a fixed bar.
- VG-02 Cable station (绳索龙门架/滑轮器械): pulley resistance, cable handle/rope, adjustable pulley height. Unified instance diagram: pulley tower with cable and handle. Action path cue: the handle follows the cable line from a high, middle, or low pulley. Beginner cue: 先找滑轮、钢索和把手/绳子；动作方向通常跟钢索拉力方向相反。 Equipment markers: pulley, visible cable, handle or rope. Example movement: Cable Lateral Raise. Common movements: Cable Lateral Raise, Lat Pulldown, Rope Pushdown. Look-for cue: Look for a cable, pulley, and handle.
- VG-03 Dumbbells (哑铃): free-weight unilateral or bilateral control with matched sides. Unified instance diagram: one or two short handheld weights. Action path cue: each hand controls its own weight path instead of sharing one fixed bar. Beginner cue: 先找一对短柄重量；左右手可以独立移动，所以更需要控制稳定。 Equipment markers: short handles, paired weights, independent sides. Example movement: Dumbbell Row. Common movements: Dumbbell Row, Dumbbell Press, Dumbbell Lateral Raise. Look-for cue: Look for one or two handheld weights.
- VG-04 Barbell (杠铃): free-weight compound lift with rack, plates, safety pins when relevant. Unified instance diagram: long straight bar with plates near a rack or platform. Action path cue: both hands share one long bar path through a squat, press, row, or hinge. Beginner cue: 先找长直杠和杠铃片；深蹲/卧推通常还需要架子和安全杆。 Equipment markers: long straight bar, weight plates, rack or platform. Example movement: Barbell Squat. Common movements: Barbell Squat, Bench Press, Deadlift. Look-for cue: Look for a straight bar, plates, and rack.
- VG-05 Machine (固定轨迹器械): guided resistance with adjustable seat, pads, handles, or lever arms. Unified instance diagram: seat, pads, handles, and lever arms. Action path cue: handles or pads move on a guided arc or fixed track after the seat is adjusted. Beginner cue: 先找座椅、靠垫、把手和调节插销；坐好后沿器械轨迹发力。 Equipment markers: seat, pads, handles or lever arms. Example movement: Machine Chest Press. Common movements: Machine Chest Press, Leg Extension, Seated Row. Look-for cue: Look for a seat, pads, handles, or lever arms.
- VG-06 Adjustable bench (可调训练凳): bench-supported pressing, rowing, split squat, hip thrust, step-up, or setup support. Unified instance diagram: flat or angled bench supporting the body. Action path cue: the bench fixes body angle or support while the limbs perform the loaded movement. Beginner cue: 先找平凳或可调斜凳；它可能支撑背、胸、臀或后脚，而不是主要阻力。 Equipment markers: flat or angled pad, bench legs, body support point. Example movement: Incline Dumbbell Press. Common movements: Incline Dumbbell Press, Chest-supported Row, Bulgarian Split Squat. Look-for cue: Look for a flat or adjustable bench.
- VG-07 Pull-up/Dip station (引体向上/双杠训练站): overhead bar, dip handles, or assisted platform for bodyweight vertical work. Unified instance diagram: tall frame with overhead bar and dip handles. Action path cue: the body moves vertically while hanging from grips or supporting on handles. Beginner cue: 先找头顶横杆、双杠把手或辅助踏板；身体重量就是主要阻力。 Equipment markers: overhead grip bar, dip handles, assist platform. Example movement: Pull-up. Common movements: Pull-up, Chin-up, Dip. Look-for cue: Look for an overhead bar, dip handles, or assist platform.
- VG-08 Resistance band (弹力带): elastic resistance anchored to a rack, door, post, or held under the feet. Unified instance diagram: elastic band anchored to a fixed point or feet. Action path cue: the band stretches as the hands or body move away from the anchor. Beginner cue: 先确认弹力带固定点可靠；越拉远阻力越大，末端要更慢控制。 Equipment markers: elastic loop or tube, secure anchor, visible stretch. Example movement: Band Face Pull. Common movements: Band Face Pull, Band Pull-apart, Banded Push-up. Look-for cue: Look for a band anchored to a rack, door, or post.
- VG-09 Leg press or hack squat (腿举/哈克深蹲器械): guided lower-body sled path. Unified instance diagram: sled or foot platform on a guided lower-body path. Action path cue: the feet push a sled or fixed platform while the body stays supported. Beginner cue: 先找大脚踏板、座椅或肩垫、安全把手；腿部沿固定轨迹推开或蹲起。 Equipment markers: large foot plate, seat or shoulder pads, safety handles. Example movement: Leg Press. Common movements: Leg Press, Hack Squat, Pendulum Squat. Look-for cue: Look for a sled platform, seat, and safety handles.
- VG-10 Bodyweight or open station (自重/开放训练空间): floor, bench, pull-up bar, dip station, bands, or open movement space. Unified instance diagram: body moving on floor, bar, bench, or open space. Action path cue: the body moves through the pattern while simple supports define the setup. Beginner cue: 如果没有明显器械，先看身体在地面、凳子、横杆或开放空间里的动作路径。 Equipment markers: floor or open space, simple support, bodyweight movement. Example movement: Push-up. Common movements: Push-up, Plank, Inverted Row. Look-for cue: Look for floor, bench, bar, bands, or open space.

Pair the visual cue with the target muscle, one setup cue, an example movement, common movements, and one look-for cue, for example: `Exercise visual guide: VG-01 Smith machine (史密斯机) | unified instance diagram: fixed bar between two vertical rails | guided incline press path | action path cue: the bar travels up and down on rails | instance diagram cue: two vertical rails with a bar that moves on a fixed track | beginner recognition cue: 先找两根竖轨和一根固定杠 | upper chest | align bench under the fixed bar path | example movement: Incline Smith Press | common movements: Incline Smith Press, Smith Squat, Smith Row | look-for cue: Look for two rails and a fixed bar`.

For app-linked plans, keep the visual guide wording consistent across the written plan, Android Exercise Visual Library, Selected Day Visual Map, Today's Exercise Visual Map, exercise thumbnail headers, the Add Exercise live equipment/action preview, and AI review context. The goal is quick recognition, not anatomical artwork: simple unified instance diagrams plus equipment/action labels are enough for a non-pro user to understand which station or setup to find in the gym.

## Movement Patterns

### Squat / Knee-Dominant

- Barbell back squat, front squat, safety-bar squat.
- Hack squat, pendulum squat, leg press.
- Bulgarian split squat, walking lunge, step-up.
- Goblet squat, dumbbell split squat.

Useful cues: brace before descent, keep pressure through midfoot, control depth, avoid knees collapsing inward, keep reps consistent.

### Hinge / Hip-Dominant

- Romanian deadlift, stiff-leg deadlift.
- Conventional or sumo deadlift for strength-biased users.
- Hip thrust, glute bridge.
- Good morning, cable pull-through, back extension.

Useful cues: push hips back, keep ribs down, maintain neutral spine, feel hamstrings load, stop range before spinal rounding.

### Horizontal Push

- Barbell bench press, dumbbell bench press.
- Machine chest press, push-up, weighted push-up.
- Cable fly, pec deck, dumbbell fly.

Useful cues: set shoulder blades, controlled touch/stretch, forearms near vertical, press without losing shoulder position.

### Vertical Push

- Barbell overhead press, dumbbell shoulder press.
- Machine shoulder press, landmine press.
- Lateral raise variations, rear-delt fly.

Useful cues: brace, avoid excessive low-back arch, move through pain-free range, lead lateral raises with elbows rather than traps.

### Horizontal Pull

- Chest-supported row, cable row, one-arm dumbbell row.
- Barbell row, machine row, inverted row.
- Rear-delt row, face pull.

Useful cues: initiate by driving elbow, pause briefly in shortened position, avoid turning rows into low-back extensions.

### Vertical Pull

- Pull-up, chin-up, assisted pull-up.
- Lat pulldown, neutral-grip pulldown, kneeling single-arm pulldown.
- Straight-arm pulldown.

Useful cues: depress shoulder blades, drive elbows down, avoid excessive torso swing, use full controlled stretch.

### Arms

- Biceps: incline curl, preacher curl, cable curl, hammer curl.
- Triceps: rope pressdown, overhead cable extension, skull crusher, close-grip press, dips if shoulders tolerate.

### Calves, Abs, Neck/Traps

- Calves: standing raise, seated raise, leg-press calf raise.
- Abs: cable crunch, hanging knee raise, reverse crunch, ab wheel, plank variations.
- Traps/neck: shrugs, farmer carries, neck work only with conservative loading and clear need.

## Substitution Rules

Substitute when equipment is unavailable, pain appears, technique is not stable, or the exercise does not match the intended target.

- Barbell squat -> hack squat, leg press, goblet squat, split squat.
- Deadlift -> RDL, hip thrust, back extension, trap-bar deadlift.
- Bench press -> dumbbell press, machine press, push-up.
- Pull-up -> pulldown, assisted pull-up, banded pull-up.
- Barbell row -> chest-supported row, cable row, machine row.
- Overhead press -> landmine press, machine press, dumbbell press.

Preserve the intended pattern, muscle target, rep range, and fatigue cost.

## Exercise Selection by User Type

- Beginners: stable lifts, fewer variations, moderate rep ranges, more technique practice.
- Intermediates: more volume distribution, weak-point isolations, planned substitutions.
- Advanced lifters: specialization blocks, fatigue-aware exercise rotation, precise set allocation.
- Limited time: machines, supersets for non-competing muscles, fewer redundant variations.
- Home gym: dumbbell, band, bodyweight, adjustable bench, pull-up bar patterns.

## Technique Red Flags

Modify or stop when:

- Pain changes mechanics.
- Range of motion becomes uncontrolled.
- The user cannot maintain bracing or joint alignment.
- Fatigue turns target work into compensation.
