# Exercise Library

Use movement patterns and equipment options instead of forcing one exercise. Choose exercises that the user can perform with stable technique, target sensation, and tolerable joints.

If the user provides equipment photos, exercise photos, or video frames, use `visual-analysis-and-food-estimation.md` for equipment identification and exercise form analysis before choosing cues or substitutions. Ask for load, reps, set number, RIR/RPE, pain, and target muscle so the visual observation can be connected to the session log.

## Exercise Visual Guide

When outputting exercises for a beginner or a user who may not recognize professional exercise names, include a concise visual guide cue that maps the name to an equipment/action category. Use the same plain categories the Android app can display:

- Smith machine: fixed bar path for guided presses, squats, or rows.
- Cable station: pulley resistance, cable handle/rope, adjustable pulley height.
- Dumbbells: free-weight unilateral or bilateral control with matched sides.
- Barbell: free-weight compound lift with rack, plates, safety pins when relevant.
- Machine: guided resistance with adjustable seat, pads, handles, or lever arms.
- Bodyweight or open station: floor, bench, pull-up bar, dip station, bands, or open movement space.

Pair the visual cue with the target muscle and one setup cue, for example: `Exercise visual guide: Smith machine | guided incline press path | upper chest | align bench under the fixed bar path`.

## Movement Patterns

### Squat / Knee-Dominant

- Barbell back squat, front squat, safety-bar squat.
- Hack squat, pendulum squat, leg press.
- Bulgarian split squat, walking lunge, step-up.
- Goblet squat, dumbbell split squat.

Useful cues: brace before descent, keep pressure through midfoot, control depth, avoid knees collapsing inward, keep reps consistent.

### Hinge / Hip-Dominant

- Romanian deadlift, stiff-leg deadlift.
- Conventional or sumo deadlift for strength-biased users.
- Hip thrust, glute bridge.
- Good morning, cable pull-through, back extension.

Useful cues: push hips back, keep ribs down, maintain neutral spine, feel hamstrings load, stop range before spinal rounding.

### Horizontal Push

- Barbell bench press, dumbbell bench press.
- Machine chest press, push-up, weighted push-up.
- Cable fly, pec deck, dumbbell fly.

Useful cues: set shoulder blades, controlled touch/stretch, forearms near vertical, press without losing shoulder position.

### Vertical Push

- Barbell overhead press, dumbbell shoulder press.
- Machine shoulder press, landmine press.
- Lateral raise variations, rear-delt fly.

Useful cues: brace, avoid excessive low-back arch, move through pain-free range, lead lateral raises with elbows rather than traps.

### Horizontal Pull

- Chest-supported row, cable row, one-arm dumbbell row.
- Barbell row, machine row, inverted row.
- Rear-delt row, face pull.

Useful cues: initiate by driving elbow, pause briefly in shortened position, avoid turning rows into low-back extensions.

### Vertical Pull

- Pull-up, chin-up, assisted pull-up.
- Lat pulldown, neutral-grip pulldown, kneeling single-arm pulldown.
- Straight-arm pulldown.

Useful cues: depress shoulder blades, drive elbows down, avoid excessive torso swing, use full controlled stretch.

### Arms

- Biceps: incline curl, preacher curl, cable curl, hammer curl.
- Triceps: rope pressdown, overhead cable extension, skull crusher, close-grip press, dips if shoulders tolerate.

### Calves, Abs, Neck/Traps

- Calves: standing raise, seated raise, leg-press calf raise.
- Abs: cable crunch, hanging knee raise, reverse crunch, ab wheel, plank variations.
- Traps/neck: shrugs, farmer carries, neck work only with conservative loading and clear need.

## Substitution Rules

Substitute when equipment is unavailable, pain appears, technique is not stable, or the exercise does not match the intended target.

- Barbell squat -> hack squat, leg press, goblet squat, split squat.
- Deadlift -> RDL, hip thrust, back extension, trap-bar deadlift.
- Bench press -> dumbbell press, machine press, push-up.
- Pull-up -> pulldown, assisted pull-up, banded pull-up.
- Barbell row -> chest-supported row, cable row, machine row.
- Overhead press -> landmine press, machine press, dumbbell press.

Preserve the intended pattern, muscle target, rep range, and fatigue cost.

## Exercise Selection by User Type

- Beginners: stable lifts, fewer variations, moderate rep ranges, more technique practice.
- Intermediates: more volume distribution, weak-point isolations, planned substitutions.
- Advanced lifters: specialization blocks, fatigue-aware exercise rotation, precise set allocation.
- Limited time: machines, supersets for non-competing muscles, fewer redundant variations.
- Home gym: dumbbell, band, bodyweight, adjustable bench, pull-up bar patterns.

## Technique Red Flags

Modify or stop when:

- Pain changes mechanics.
- Range of motion becomes uncontrolled.
- The user cannot maintain bracing or joint alignment.
- Fatigue turns target work into compensation.
