# Plan Optimization

Use this when the user already has an existing plan, spreadsheet, training log, app export, or coach-written routine and wants it improved.

## Existing Plan Audit

Audit the existing plan before rewriting it:

- Goal fit: does the plan match hypertrophy, fat loss, strength support, recomposition, or maintenance?
- Schedule fit: can the user complete the plan with available days, time, equipment, and recovery?
- Muscle balance: are weekly hard sets distributed across priority and non-priority muscles?
- Movement pattern coverage: squat, hinge, horizontal push, vertical push, horizontal pull, vertical pull, calves, trunk, arms as appropriate.
- Exercise-level quality: does each movement have a reason, target muscle, rep range, progression rule, and substitution?
- Fatigue cost: are high-fatigue compounds, failure sets, cardio, and life stress recoverable?
- Redundancy: are multiple exercises doing the same job while weak points are missed?
- Progression decision: does the plan say when to add reps, load, sets, or deload?
- Safety: does pain, injury history, or equipment risk require modification?

## Optimization Workflow

1. Preserve what is working.
2. Remove redundant or low-stimulus work.
3. Add missing movement patterns or weak point work.
4. Align volume with training age and recovery.
5. Clarify progression and deload rules.
6. Convert vague prescriptions into executable sessions.
7. Define what data will decide the next adjustment.

Do not replace a plan just to make it look different. Improve the bottleneck.

## Volume Audit

For each major muscle, estimate weekly hard sets. Count a set when:

- It targets the muscle meaningfully.
- It is close enough to failure to create a hypertrophy stimulus.
- Technique and range are acceptable.

Use indirect set counting conservatively:

- Bench press counts strongly for chest and triceps/front delts secondarily.
- Rows count strongly for back and secondarily for biceps/rear delts.
- Squats count strongly for quads/glutes depending execution.

Avoid pretending set counts are exact. Use them as coaching signals.

## Common Fixes

- Too much volume and poor progress: remove low-quality sets, reduce failure exposure, add deload.
- Too little volume and good recovery: add 1-2 hard sets per priority muscle per week.
- Poor exercise fit: swap while preserving muscle target and movement pattern.
- No progression: add double progression, load progression, or set progression.
- Weak point lagging: move it earlier, increase frequency, or improve exercise selection.
- Fat-loss recovery poor: keep intensity, reduce volume/cardio overlap, protect protein and sleep.

## Optimized Plan Output

When optimizing, show:

- What to keep.
- What to change.
- Why each change matters.
- New weekly structure.
- Exercise-by-exercise prescriptions.
- Volume summary by muscle.
- Tracking and progression decision rules.
